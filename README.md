# Portfólio de Introdução ao Python 🐍

Este repositório é parte de um projeto educacional desenvolvido pela equipe **Tech Bridge** para demonstrar os conceitos básicos da linguagem Python. Aqui você encontrará exemplos práticos de como usar estruturas de controle, funções, operadores e entrada de dados.

## Conceitos Abordados

- 📥 `input()` – entrada de dados
- 🔀 `if`, `elif`, `else` – condições
- 🔁 `for`, `while`, `range()` – laços de repetição
- 🧮 Operações matemáticas
- 🧩 `def` – criação de funções

## Estrutura

- **exemplos-basicos/**: pequenos scripts que demonstram cada conceito individualmente.
- **mini-projetos/**: desafios que combinam os conceitos para resolver problemas simples.

## Como executar

1. Clone este repositório:
```bash
git clone https://github.com/seu-usuario/python-intro-portfolio.git
cd python-intro-portfolio
```

2. Execute os scripts com Python:
```bash
python exemplos-basicos/input_exemplo.py
```
